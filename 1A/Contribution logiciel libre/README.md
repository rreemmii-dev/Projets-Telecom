# Contribution à un logiciel libre

Ce projet consiste à contribuer à un logiciel libre, dans mon cas [Anki](https://github.com/ankitects/anki).

Le morceau de code édité était en Rust, mais des parties importantes du projet étaient en Python et en JavaScript.


## Table des matières

* [Tâches à réaliser](#tâches-à-réaliser)
* [Apprentissages](#apprentissages)


## Tâches à réaliser

Le travail s'est déroulé en plusieurs parties :

### Trouver un logiciel intéressant

La première étape consiste à trouver plusieurs logiciels open-source que nous utilisons personnellement et apprécions.
Il faut également que le logiciel soit activement maintenu.

### Etude du repository GitHub et premiers contacts

Durant cette étape, il faut trouver plusieurs demandes de contribution (= "issues GitHub") qui semblent réalisables.
C'est également l'occasion d'un premier contact grâce à une contribution dans la documentation.

### Ecriture de la contribution

Enfin, il reste à réaliser la contribution parmi celles précédemment sélectionnées.
Dans mon cas, les demandes de contribution ont été acceptées (https://github.com/ankitects/anki/pull/3912 et https://github.com/ankitects/anki/pull/3944).


## Apprentissages

Ce projet m'a principalement permis de travailler dans une grande base de code, écrite par une autre personne que moi.
Il faut notamment réussir à comprendre comment le code est organisé avant de le modifier, d'où l'importance d'avoir un code propre.
